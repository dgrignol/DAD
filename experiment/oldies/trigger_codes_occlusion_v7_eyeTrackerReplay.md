# Trigger Codes (MoveDot1_experiment_occlusion_v16_eyeTrackerReplay)

| Code / Range | Short label | Extended description |
| --- | --- | --- |
| 31 | Trial onset (always_visible) | First frame of an `always_visible` trial. |
| 41 | Trial onset (occluded_nondeviant) | First frame of an `occluded_nondeviant` trial. |
| 51 | Trial onset (occluded_deviant) | First frame of an `occluded_deviant` trial. |
| 1..N | Trial sequence identity | Emitted once at fixed frame offset (`Conf.sequenceIdentityTriggerFrame`, default 12). Value equals `xySeqs(source_index).sequence`. |
| 111 | Occlusion start | First frame where occlusion starts (`occlusion_start_frame`). |
| 112 | Occlusion complete | First frame where the dot is fully hidden (`occlusion_complete_frame`). |
| 114 | Occlusion end start | First frame where the dot starts reappearing (`occlusion_end_frame`). |
| 115 | Occlusion end complete | First frame where the dot is fully visible again (`occlusion_end_complete_frame`). |
| 116 | Catch type-1 disappear | Type-1 catch only: first frame where dot disappears in run 1. |
| 117 | Catch type-1 reappear | Type-1 catch only: first frame where dot reappears in run 1. |
| 118 | Catch question start | First frame of catch question prompt. |
| 101 | Catch question end | Emitted when catch question closes (response or timeout). |
| 201 | Catch response YES | Catch response code YES (`changed`). |
| 202 | Catch response NO | Catch response code NO (`not changed`). |
| 113 | Catch timeout | No response during catch-response window. |
| 150 | Gaze break | Emitted when fixation leaves the acceptance window for `fixBreakToleranceFrames` consecutive frames. |
| 151 | Replay start | Emitted once per run immediately before the first replayed trial starts after at least one gaze break. |

## Notes

- This map is specific to `experiment/MoveDot1_experiment_occlusion_v16_eyeTrackerReplay.m`.
- Runtime validates that dynamic sequence trigger range `1..N` does not overlap fixed codes.
- Catch planning is generated by `experiment/CreateInputFiles_v19_threeRunsPerBlock_catch_eyeTrackerReplay.m`.
- Eye-tracker integration modes:
  - Real EyeLink mode can emit 150/151 from measured gaze samples.
  - Fake eye-tracker mode can emit 150/151 from simulated breaks (`fakeGazeBreakRate`).
  - If eye tracking is disabled, codes 150/151 are never emitted.
